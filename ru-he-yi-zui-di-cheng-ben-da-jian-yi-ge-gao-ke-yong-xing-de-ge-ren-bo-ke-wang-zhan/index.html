如何以最低成本，搭建一个高可用性的个人博客网站

2021-08-25

准备工作

在开始本教程之前，我们要开始点准备😝：

  1. 一个邮箱！（这是必须的！！！😡）
  2. 一个域名（无需备案）（可不准备，但是会导致网站可用率大幅度下降 ）
  3. 一台可以正常连接到GitHub的电脑！（如果无法访问或者间断性无法访问，建议科学上网，否则会影响接下来的网站配置）
  4. 一个Gridea的安装文件（可在https://gridea.dev/#started下载属于自己操作系统的安装文件）

正式开始

🙄这里忽略Gridea的安装教程！！

第一步 注册一个GitHub账号😲

  1. 打开Https://GitHub.com/
  2. 点击右上角的SignUp进入注册页面



    在输入完邮箱（早就让你准备了！😟）、密码、用户名和确认的字母Y后，输入发送到你邮箱的验证码完成github的注册！！！😎😎😎
    然后他会询问你的年龄和感兴趣的仓库以及你的仓库计划（前两个按照真实的填就可以了，仓库计划选择free！！但是如果你有钱请随意！😐）

第二步 创建一个仓库并且获取到GitHub的Token！😋

  1. 点击右上角的加号，并且选择第一个选项，进入创建仓库页面
  2. 创建仓库，存放构建后的网站文件。创建一个公开仓库，Repository name为用户名.github.io（将用户名替换为你的 GitHub 用户名，必须这么填！！！），并且选择Public及下方的Add a README file。

    完成上方的配置后，你将会得到以下页面
  3. 接下来创建一个仓库 Token
    点击https://github.com/settings/tokens/new创建一个 Token。
    Note随意填，Expiration选择No Expiration（即永不过期），勾选上 repo 的相关权限

    创建后，你将会得到以下页面

    复制Token备用！！！！！一旦关闭网页将不能再看到它！！！

这样我们就创建了一个仓库并且获取到了GitHub的Token。🙏🙏

第三步 配置Gridea，使其与GitHub通讯！

进入 Gridea 进行远程设置。

  1. 域名输入用户名.github.io
  2. 仓库输入用户名.github.io
  3. 分支输入main
  4. 用户名和Token输入对应的即可
    （如果你想使用自己的域名，你需要将域名与CNAME改成自己的域名信息！！，然后将自己的域名解析到用户名.github.io）
    然后点击「检测远程链接」，进行配置检查😃😃
    如果提示远程连接成功即可点击保存！

    紧接着点击 Gridea 中的同步按钮即可。初次部署等待几分钟之后就可以去浏览器访问了，默认地址为你所填写的域名。（成功访问的时候是不是有一点的小激动呢😂😂，如果显示无法访问网页，试着多刷新几次！！或者等待一会在刷新！😑😑毕竟同步也是需要时间的！）
    别放弃！因为还有好长一段路呢！！😛😛

第四步 选择一个适合自己的主题并且通过jsdelivr加速GitHub部署的CSS文件以提高访问速度！

Gridea官方提供了几款默认的主题，如果你需要更多的主题，请访问https://gridea.dev/themes
下载后解压在你的Gridea安装目录下的themes文件夹里，重启Gridea即可在主题 》选择主题使用！

（选择完主题并且进行配置之后，别忘了点击左下角的同步，否则网页是不会变动的！✍️✍️✍️这点非常重要，每次修改完配置，或写完文章，都要点击同步，然后等待2-5分钟刷新网页，才会有变更的效果！！）😏😏😏
这里以官方内置的主题simple为例，使用jsdelivr加速主题css文件！
首先你要在Gridea的主题设置里面，设置头像与网页图标，同步并等待2-5分钟，刷新网页，直到网页显示新图片为止。
紧接着在网页上右键打开查看网页源代码，你会发现三个以用户名.github.io或你自己域名开头的开头的网址，他们分别是一个以main.css结尾的css文件，与两个中间掺杂着picture或image的的图片文件。
复制以main.css结尾的css文件的链接
打开https://www.jsdelivr.com/github

在第一栏输入刚才复制的文件链接，第二栏将自动得出加速后的链接地址，复制备用😶
紧接着打开Gridea的安装目录，打开themes下的simple文件夹（不同主题的文件夹名称不一样，这里以simple主题作为参考），然后进入templates文件夹 》_blocks文件夹

然后打开head.ejs(使用记事本打开即可)
将其中的<%=themeConfig.domain%>/styles/main.css替换为刚才复制的加速链接，Ctrl+S保存并且关闭窗口。
